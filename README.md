# Frontend

Frontend base con React, TypeScript y Tailwind para la primera fase operativa.

## Incluye

- login para un unico admin.
- formulario de configuracion Gmail.
- prueba de conexion.
- accion de sincronizacion manual.
- ticker en tiempo real por WebSocket con correos aprobados.

## Arranque

```bash
npm install
npm run dev
```

El frontend asume backend en `http://localhost:3000`.
